# Security Policy

## Reporting a vulnerability

This repository does not contain the [official release version of Notepad++](https://github.com/notepad-plus-plus/notepad-plus-plus/). It is a modification that **does not alter the source code!** All changes have been made to publicly accessible files and do not affect hidden, unreadable, or binary files.

If you believe you have found a security vulnerability in this repository you
can report it to the developer Don HO using [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability).

If you are unable to file a report via the above mechanism or want to report
vulnerabilities, please email developer **[don.h@free.fr](mailto:don.h@free.fr)**

**Please do not report security vulnerabilities through public GitHub issues,
discussions, pull requests or any other publicly accessible format!**

Thanks for helping make Notepad++ safe for everyone.
